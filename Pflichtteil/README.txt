Projekt: WebXR + A-Frame Demo
Enthaltene Dateien:
- inline-session.html   : kombinierte Seite mit AR- und VR-Buttons (WebXR / Three.js)
- aframe-scene.html     : A-Frame Szene mit 5 Objekten, geladenem glTF-Modell und Textur, Licht + Schatten
- demo.gif              : kleines kurzes Demo-Video (animiertes GIF) das beide Dateien erklärt
Anleitung:
- Öffnen Sie die HTML-Dateien lokal im Browser (Chrome/Edge mit WebXR-Unterstützung für AR/VR).
- aframe-scene.html benötigt Internetzugang, da das glTF-Modell von einem CDN geladen wird.
- Wenn Sie die Seiten in VS Code betreiben: nutzen Sie die Live Server Erweiterung oder öffnen die Dateien per Datei->Öffnen im Browser.
